# Proyecto Integrador - Generador Seguro de Contraseñas

## Asignatura
Lógica de Programación

## Estudiante
Isaac Arias

## Objetivo del sistema
Desarrollar un programa en Python que permita generar contraseñas seguras según las preferencias del usuario, aplicando los conocimientos adquiridos durante la asignatura.

## Funcionalidades
- Solicita la longitud de la contraseña.
- Permite incluir mayúsculas.
- Permite incluir números.
- Permite incluir símbolos especiales.
- Genera una contraseña aleatoria y segura.

## Tecnologías utilizadas
- Python
- Visual Studio Code
- GitHub
